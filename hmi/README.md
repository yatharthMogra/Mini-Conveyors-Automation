# HMI Design

Design specifications for the CODESYS Visualization screens.

## Screens

| Screen | Description |
|---|---|
| Main Overview | Conveyor graphic, sensor status, Start/Stop/Fault Clear buttons |
| Alarms & Status | Active alarm list, system state indicator |
| Metrics Dashboard | Box count, cycle time, jams, throughput, parameter inputs |

## Implementation

HMI screens are built using CODESYS Visualization objects within the CODESYS IDE. The design documents in this folder specify layouts, tag bindings, and behavior for each screen.
